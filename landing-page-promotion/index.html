Run it yourself in 60 Seconds

Zero trust required. The code is open source and runs entirely on your machine.

1

Prerequisites

Ensure you have Node.js (v18+) and Git installed on your computer.

2

Clone the Repository

Download the source code from GitHub to your local machine.

3

Install & Run

Install dependencies and start the local development server. The dashboard will open in your browser automatically.

Security Note: When the app launches locally (localhost:3000), you will input your seed phrase or private keys. Since this runs on your machine, your keys never leave your computer.

Terminal

➜ ~ git clone https://github.com/nullset/nullset.git

➜ ~ cd nullset

➜ nullset npm install

➜ nullset npm run dev
